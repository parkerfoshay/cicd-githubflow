# What is NoSQL?

NoSQL is an abbreviation of "Not only SQL".

Unlike relational databases, NoSQL databases do not rely on a tabular structure.

The main kinds of NoSQL databases are:

 - document (What MongoDB is)
 - key-value
 - wide-column
 - graph
